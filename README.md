# POS Printer Library

A multiprotocol Go library for thermal printer control.

## Installation

```bash
go get github.com/AdConDev/pos-printer
```

## Usage

```go
import posprinter "github.com/AdConDev/pos-printer"
```

## Features

- Multiple protocol support
- Multiple connection types
- Profile management
